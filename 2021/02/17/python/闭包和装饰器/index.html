闭包和装饰器

闭包

闭包的概念：

一个完整的闭包须满足以下三个条件

  • 1.函数中嵌套一个函数
  • 2.外层函数返回内层函数的函数名
  • 3.内层函数对外部作用域有一个非全局变量的引用(不能引用全部变量)保持封闭状态。

闭包函数调用:返回的不仅仅是个函数，这个函数还有一个封闭的外部作用域

闭包的作用就是锁定数据 保持数据安全

装饰器

开放封闭原则

开放：对代码功能的扩展是开放的
封闭：对已经实现的功能代码修改是封闭的

装饰器是对开放封闭原则的一种体现

装饰器的作用：在不修改功能代码的同时，给代码扩展新的功能

@是装饰器的语法 @ddt @data @classmethod
